#  Tic-Tac-Toe GUI Game (Python)

A full-screen Tic-Tac-Toe game built with **Python (Tkinter)** featuring:
- One-window GUI (no pop-ups)
- Player name input
- 5-round match system
- Highlighted winning lines
- Turn animation
- Sound effects
- Fullscreen mode (F11 / Esc)

This project was developed as a learning and portfolio project.
